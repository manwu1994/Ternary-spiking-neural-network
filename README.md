# Ternary-spiking-neural-network
Ternary spiking neural network
TSNN architecture achieves up to 16x model compressions and achieves 98.43%, 89.07%, 65.24% accuracy with 4 time steps, respectively on N-MNIST, CIFAR-10, CIFAR-100, respectively.
